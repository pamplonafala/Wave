# Wave
Creates a non-visible wavey matrix

### How to use
To see an especific place within the matrix use:
```js
wave.xy(<x>, <y>, true);
```

To apply wave movement in objects use:
```js
obj.xWave = wave.xy(obj.x, obj.y, true);
```
